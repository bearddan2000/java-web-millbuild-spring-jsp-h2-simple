# java-web-millbuild-spring-jsp-h2-simple

## Description
A jsp springboot java gradle build,
that connects to in memory h2 database.

## Tech stack
- springboot
  - jsp
- millbuild
  - h2 drivers
  - lombok
  - PostConstruct annotation
- bootstrap
- jquery
- dataTable

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
